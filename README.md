# Front-end
Update node-module
```bash
yarn
```

Run project
```bash
yarn dev
```
[Open on browser: http://localhost:5173/](http://localhost:5173/)


# Backend
Update composer
```bash
composer update
```

Run project
```bash
php -S localhost:8000
```
[Open on browser: http://localhost:8000/](http://localhost:8000/)
